What is AutoImage?

AutoImage is an AI image generation platform that provides access to multiple AI models including DALL-E 3, Flux, Google Imagen 3, and various Stable Diffusion variants. The platform combines these models with automation features for bulk image generation.

The main features are the dynamic variable system and bulk generation capabilities that allow creating multiple unique variations from a single prompt template.

Auto Image Generation

The Auto Image feature automatically generates multiple images from your prompt without manual intervention. Each image uses different variable combinations to create variety.

How Auto Generation Works:

  • Set your maximum number of images (1-10)
  • Enable the "Auto" checkbox
  • Click START to begin automatic generation
  • Each image uses different variable combinations for variety

AI Prompt Enhancement

The Enhance feature uses OpenAI's GPT models to automatically improve your prompts by adding professional photography terms, artistic styles, and technical details that result in higher-quality images.

How AI Enhancement Works:

Dynamic Variable System: ${variable}

The ${variable} system allows you to create dynamic prompts that automatically generate unique variations by replacing variables with random selections.

How Variables Work:

Basic Variables: ${cat} → Randomly selects from: Maine Coon, Siamese Cat, Bengal Cat, etc.
Persistent Variables: $${cat} → Same selection used across all images in a batch
Custom Variables: custom_color=red,blue,green

Example Variable Usage:

Input: A ${cat} in ${hip chill scenes} art style
Output 1: A Maine Coon in Urban rooftop party art style
Output 2: A Siamese Cat in Cozy coffee shop art style

Enhancement Options: Multiplier, Mixup, Mashup

These powerful options let you add variety and creativity to your image generation process:

Multiplier

Adds text to each prompt part

Input: "A cat, A dog" + "lighting"
Output: "A cat, lighting, A dog, lighting"

Mixup

Reorders comma-separated parts

Input: "A cat, A dog, A bird"
Output: "A bird, A cat, A dog"

Mashup

Shuffles individual words

Input: "A cat, A dog"
Output: "A dog cat A"

Getting Started

To begin using AutoImage:

Basic Steps:

  1. Create an account and add credits
  2. Select a model based on your quality and cost requirements
  3. Write a prompt with variables like ${subject} in ${style}
  4. Enable Auto generation and set your maximum number
  5. Click START to begin generation
